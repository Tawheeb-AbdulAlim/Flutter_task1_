# النشاط الأول: شاشة تسجيل الدخول (Login Screen)

هذا المشروع هو تطبيق Flutter بسيط يحتوي على شاشة تسجيل دخول تفاعلية.

 الميزات:
- واجهة مستخدم منظمة باستخدام Column و Padding.
- حقول إدخال نصية (TextField) للبريد الإلكتروني وكلمة المرور.
- استخدام TextEditingController للحصول على البيانات.
- زر دخول يعرض رسالة تأكيدية باستخدام SnackBar.
